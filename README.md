### 用法

1.  请确保已安装node环境，如果没有安装，请去 [nodejs官网](https://nodejs.org)下载安装
2.  执行以下代码

``` bash
$ npm install -g vue-cli (第一次使用时执行，只用安装一次)
$ vue init zhipenglin/mobile-app my-project
$ cd my-project
$ npm install
$ npm run start
```
